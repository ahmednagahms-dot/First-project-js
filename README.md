# 🛒 ShopEase - E-Commerce Web Application

ShopEase is a dynamic, modern, and fully responsive multi-page e-commerce web application. Built entirely with vanilla JavaScript, HTML5, and CSS3, it offers users a smooth shopping experience with full control over browsing, searching, and managing their favorite items.

🎨 **Live Demo:** [View ShopEase Live](https://ahmednagahms-dot.github.io/First-project-js/)

---

## 🚀 Features

- **🛍️ Product Browsing:** View a dynamic list of items with clear pricing, categories, and ratings.
- **🔍 Advanced Search & Filter:** Search for your favorite items seamlessly by **Product Name** or filter down by specific **Categories**.
- **🛒 Interactive Shopping Cart:** Add items to your cart, dynamically calculate totals, and manage quantities in real-time.
- **❤️ Favorites System:** Save items you love to a personalized favorites list for easy access later.
- **🔐 Authentication Pages:** Dedicated user workflows with fully structured `Login` and `Register` screens.
- **📱 Responsive Design:** Tailored layout optimized for desktops, tablets, and smartphones.
