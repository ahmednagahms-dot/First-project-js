# 🛒 ShopEase - E-Commerce Web Application

ShopEase is a dynamic, modern, and fully responsive multi-page e-commerce web application. Built using modern web technologies, it offers users a smooth shopping experience with full control over browsing, searching, and managing their favorite items.

🎨 **Live Demo:** [View ShopEase Live](https://ahmednagahms-dot.github.io/First-project-js/)

---

## 🚀 Features

- **🛍️ Product Browsing & Details:** View a dynamic list of items with dedicated layouts for single product views (`card.html`).
- **🔍 Advanced Search & Filter:** Search for your favorite items seamlessly by **Product Name** or filter down by specific **Categories**.
- **🛒 Interactive Shopping Cart:** Add items to your cart, dynamically calculate totals, and manage quantities in real-time.
- **❤️ Favorites System:** Save items you love to a personalized favorites list for easy access later.
- **🔐 User & Authentication Pages:** Dedicated user dashboard (`user.html`) and fully structured `Login` and `Register` workflows.
- **📱 Responsive Design:** Sleek, modern, and completely responsive UI built with Tailwind CSS.
